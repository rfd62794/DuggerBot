# ADR-004: n8n is Permanently Tombstoned

**Status:** Accepted
**Date:** 2026-06-15
**Author:** Robert Floyd Dugger

## Context

n8n was evaluated as an orchestration layer in PrivyBot. The TG-MCP architectural decision made it obsolete. It was tombstoned.

## Decision

n8n is never installed on Tower. Never evaluated again. Never added to DuggerBot in any form.

## Consequences

- Any proposal to add n8n is rejected without review.
- This is not an open question.
