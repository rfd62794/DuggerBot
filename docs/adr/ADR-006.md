# ADR-006: DuggerBot is Personal Infrastructure — Not a Platform

**Status:** Accepted
**Date:** 2026-06-15
**Author:** Robert Floyd Dugger

## Context

OpenClaw became a platform with a marketplace (ClawHub), 20+ channels, and enterprise features. This is not the goal for DuggerBot.

## Decision

DuggerBot serves one person. It has one Telegram channel. It has no skills marketplace. It has no multi-user support. It has no public API. Features are added only when Robert hits a specific, real friction. Not before.

## Consequences

- Any directive that adds multi-user support, a skills marketplace, additional messaging channels (beyond Telegram), or a public-facing API violates this ADR.
- The scope is intentionally narrow.
